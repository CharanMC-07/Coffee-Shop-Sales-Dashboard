# ☕ Coffee Shop Sales Dashboard

## 📌 Overview
This project is an interactive Power BI dashboard created to analyze coffee shop sales performance. It provides insights into revenue, profit, transactions, quantity sold, branch performance, and product categories.

---
## 📊 Features
- Total Revenue, Profit, Transactions, and Quantity Sold KPIs
- Branch-wise sales analysis
- City-wise revenue tracking
- Product category performance
- Interactive filters for Year, Month, and Shop Name
---
## 🛠️ Tools Used
- Power BI
- DAX
- Power Query
---
## 📂 Dataset Includes
- Date
- Product Category
- Quantity Sold
- Revenue
- Profit Margin
- Shop Name
- Branch
- City
---
## 🚀 How to Use
1. Open the `.pbix` file in Power BI Desktop
2. Refresh the dataset
3. Use filters and interact with visuals
